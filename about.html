Hello! Welcome to

Pure Javascript Geotagging tool

You are able to check the position of your geotagged images. Simply drag the images to the map and they will be positioned on the place they were shooted. You can drag several images simultaneously. If your images do not have a geotag, you are able to position them on the map and save the image with position stored in the file.

Technical Notes! This tool uses HTML5 File API. It has been tested with browsers supporting this API. It works with latest versions of Firefox and Chrome browser. The Opera browser needs confirmation on uploading the file and drops untagged images to 0,0 position. I could not make it working for Internet Explorer 10 as the files property of the dataTransfer object is always undefined. There is no presently a library for storing jpeg headers using javascript only and it cannot be made in 100 lines of code. That is why when you download the image with geotag it will contain only GPS data and all the other tags will be gone. So please do not overwrite your original files and wait for complete version of this tool which will fully support jpeg header modification!
